# 答辩功能代码索引

最后整理时间：2026年3月31日

## 1. 文档用途

这份文档用于答辩现场快速回答四类问题：

1. 这个功能对应哪个页面。
2. 这个页面调用了哪个接口。
3. 这个接口落到哪个后端业务类。
4. 这个功能最终写入哪些核心表。

推荐查找顺序：

1. 先看“页面路由”和“前端页面文件”。
2. 再看“前端 API”和“后端接口”。
3. 最后看“后端入口”和“核心表”。

## 8. 老师常问问题的最快定位

### 如果老师问“下单功能在哪里”

1. 路由：`/order-confirm`
2. 页面：`agricultural-frontend/src/views/user/OrderConfirm.vue`
3. API：`createOrder`
4. 接口：`POST /user/orders`
5. 后端：`UserOrderController -> UserOrderService.create`

### 如果老师问“支付流程在哪里”

1. 路由：`/order-pay/:orderId`
2. 页面：`agricultural-frontend/src/views/user/OrderPay.vue`
3. API：`payOrder`
4. 接口：`POST /user/orders/{id}/pay`
5. 后端：`UserOrderController -> UserOrderService.pay`

### 如果老师问“扫码溯源在哪里”

1. 路由：`/trace/:traceCode`
2. 页面：`agricultural-frontend/src/views/user/TraceArchive.vue`
3. API：`getTraceArchive`
4. 接口：`GET /user/traces/{traceCode}`
5. 后端：`UserProductController -> UserProductService.getTraceArchive`

### 如果老师问“商家评分和评论聚合在哪里”

1. 路由：`/shop/:merchantId`
2. 页面：`agricultural-frontend/src/views/user/MerchantShop.vue`
3. 接口：`GET /user/merchants/{id}`、`GET /user/merchants/{id}/comments`
4. 后端：`UserMerchantController -> UserMerchantService.getShop/comments`

### 如果老师问“高并发下一致性怎么保证”

建议直接讲这 5 个点：

1. `UserOrderService.create`
2. `ProductInfoMapper`
3. `UserOrderService.pay`
4. `AdminPaymentService.updateRefundStatus`
5. `OrderTimeoutJob`
