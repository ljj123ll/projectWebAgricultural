# 助农电商平台前端 - 运行说明

## 项目概述

本项目是基于 Vue 3 + Vite + Pinia + Element Plus + TypeScript 开发的助农电商系统前端，包含用户端、商家端和管理员端三个子系统。

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **状态管理**: Pinia + pinia-plugin-persistedstate
- **UI 组件库**: Element Plus
- **开发语言**: TypeScript
- **样式预处理**: SCSS
- **路由管理**: Vue Router 4
- **图标**: Element Plus Icons

## 项目结构

```
src/
├── apis/              # API 接口封装
├── assets/            # 静态资源
├── components/        # 公共组件
├── layout/            # 布局组件
│   ├── UserLayout.vue      # 用户端布局
│   ├── MerchantLayout.vue  # 商家端布局
│   └── AdminLayout.vue     # 管理员端布局
├── router/            # 路由配置
│   └── index.ts
├── stores/            # Pinia 状态管理
│   └── modules/
│       ├── user.ts    # 用户状态
│       ├── cart.ts    # 购物车状态
│       ├── order.ts   # 订单状态
│       └── message.ts # 消息状态
├── styles/            # 全局样式
│   └── common.scss
├── types/             # TypeScript 类型定义
│   └── index.ts
├── utils/             # 工具函数
│   └── request.ts     # axios 封装

    ├── merchant/      # 商家端页面
    └── admin/         # 管理员端页面
```

## 主要功能模块

### 用户端 (/)
- 首页：商品浏览、搜索、筛选、热销榜、滞销专区、助农新闻
- 商品详情：多图轮播、溯源信息、加购/立即购买
- 购物车：商品管理、数量修改、结算
- 订单：确认订单、支付、订单列表、订单详情
- 个人中心：个人信息、收货地址、我的评价
- 售后：申请售后、售后沟通

### 商家端 (/merchant)
- 控制台：待办事项、核心数据
- 店铺信息：基础信息维护、资质上传、收款账户
- 商品管理：增删改查、溯源二维码生成
- 订单管理：订单处理、物流填写
- 售后处理：售后申请处理
- 数据统计：销量统计、营收分析
- 对账与补贴：对账明细、补贴查看

### 管理员端 (/admin)
- 数据看板：平台核心数据统计
- 用户管理：用户列表、商家审核
- 商品审核：商品上架审核
- 订单管理：全平台订单管理
- 资金管控：打款管理、退款处理
- 评论管理：评论审核、违规删除
- 新闻管理：助农新闻发布
- 系统运维：日志查看、数据备份

## 常见问题

### 1. 安装依赖失败
尝试使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

### 2. 端口被占用
修改 `vite.config.ts` 中的 server 配置：
```ts
server: {
  port: 3000
}
```

### 3. 代理配置
在 `vite.config.ts` 中配置代理：
```ts
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true
    }
  }
}
```
